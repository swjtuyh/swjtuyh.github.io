hexo+github搭建个人博客

2018-06-16
blog

一直想搭个自己的blog，尝试过几次都失败后，经过不懈的努力和参考网上各大神的教程，终于成功搭建了自己的博客。现在将自己搭建的过程记录下来，做个记录，也希望可以帮到哪些想自己搭建博客的人。

注册github

搭建个人博客，需要一个github账号，注册完账号后，创建一个新的仓库，仓库的名字为”你的账户名.github.io”

主题选择

搭建自己的博客，需要去Hexo选择自己喜欢的主题，选择主题后，可以看到主题在github上的wiki，wiki中有详细的本主题的使用的说明，配置的信息。可以将选择好的主题clone的自己的github中。

注意，主题clone不要放在上面新建的仓库中，此仓库用来做你的博客的仓库

Hexo设置

你在hexo上选中的主题并不是你的博客文件，只是一个主题。需要创建一个blog，然后将你选中的主题替换掉博客创建时的默认主题。

具体操作可以参照hexo官网的操作指南，也可采用一次一下链接中的说明tianbo_zhang的专栏

修改配置

使用hexo命令可以创建出一个博客文件，此博客文件此时在你的电脑上，我们需要修改博客文件中的_config.yml文件

git@github.com/username/username.github.io.git
1
deploy: # 部署相关配置
2
type: git # 使用 Git 提交
3
repository: git@github.com/username/username.github.io.git# 博客仓库地址
4

#repository: https://github.com/xxx/xxx.github.io.git
修改完配置文件后，你需要将你创建的文件push的你的github仓库中，仓库地址为上面代码中repository中给出的路径。

至于增加将本地文件push到github中，请参考下面的链接如何将本地文件上传到Github上

将文件上传到github仓库中后，你的个人博客就已经可以访问了。方位地址为https://username.github.io，

如果你想使用自己注册的域名去访问的话。需要去购买一个域名，并进行备案，备案完成后可以将github网站的域名解析到你购买的域名中，即可进行访问。
